# Loops 
        for each loop
        ``` 
        for(int num : nums)
        {
            System.out.println(num);
        }
        ```
        For loop
        ```
        for (int i = 0; i < nums.length; i++)
        {
            System.out.println(num[i]);
        }
        ```
